# MEMORY.md - 长期记忆

## 工作流程

### 每日工作日志
**触发时机：** 用户说"再见"、"晚安"、"今天就到这"等结束语时

**操作步骤：**
1. 回顾当天的对话和完成的任务
2. 整理成条目列表
3. 保存到 `daily-logs/YYYY-MM-DD.md`
4. 给用户一个简短的总结确认

**存储位置：** `workspace/daily-logs/`

---

## 工作流程规范

### 任务文件组织
**规则：** 每个独立任务创建专属文件夹，每个版本一个子文件夹

**命名规范：**
- 文件夹名：小写英文+短横线
- 版本子文件夹：`任务名_v1`, `任务名_v2`, ...
- 新版本不覆盖旧版本
